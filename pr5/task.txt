Практическое задание
Выполнить визуализацию многомерных данных, используя t-SNE.
Необходимо использовать набор данных MNIST или fashion MNIST (можно
использовать и другие готовые наборы данных, где можно наблюдать
разделение объектов по кластерам, запрещается использовать данные,
которые используются в файле для примера). Рассмотреть результаты
визуализации для разных значений перплексии.
Выполнить визуализацию многомерных данных, используя UMAP с
различными параметрами n_neighbors и min_dist. Рассчитать время работы
алгоритма с помощью библиотеки time и сравнить его с временем работы t-SNE.
Оформить отчет о проделанной работе. Отчет должен содержать
результаты визуализации для разных значений параметров и выводы
